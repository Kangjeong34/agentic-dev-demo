## 회귀 범위
- direct: 조각 생성·이동·회전·낙하·고정·점수·레벨·게임오버 (신규)
- shared: Board.clearFullLines / rows (기존 줄 삭제 로직 — 시그니처·동작 보존)
- evidence: 기존 BoardTest 3개 테스트가 변경 없이 통과

## 완료 정의
- 완료는 사람 느낌이 아니라 `./gradlew test`(자바) 통과로 판정한다.
- UI/입력/렌더링은 이번 범위 밖이며, 모든 동작은 결정적 JUnit 게이트로만 검증한다.
